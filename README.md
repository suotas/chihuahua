# chihuahua

chihuahua is BTC trade automation system.

# Getting started
```git clone https://github.com/suotas/chihuahua.git```  
```cp example.env .env```  
```vi .env```

enter your bitbank API key at BITBANK_API_KEY.  
enter your bitbank API secret at BITBANK_SECRET.

```go run main.go```
